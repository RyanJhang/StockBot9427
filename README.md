## install pipenv
pipenv 是用來管理python module，也可以說成管理pip有點像虛擬環境，是為了應對不同專案可能會需要不同的環境設定module，所延伸出的虛擬pip管理
```
安裝pipenv
pip install pipenv
```
產生python3的虛擬環境
pipenv --three
```
可以進入虛擬機環境
pipenv shell
```
